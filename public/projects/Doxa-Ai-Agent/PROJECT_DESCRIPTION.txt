Doxa AI Agent
An intelligent support ticket management system and AI agent designed to automate and streamline regulatory and operational inquiries using RAG.

Tech Stack:
- Next.js 16.1 (App Router)
- FastAPI / Python
- TypeScript 5
- Tailwind CSS 4
- ChromaDB (Vector Search)
- Mistral AI Integration
- JWT Authentication

Project Description:
This project showcases a complete full-stack architecture featuring an AI-driven orchestration engine. It automates the processing of user tickets and support requests by analyzing regulatory documents (DOXA) and generating intelligent, context-aware responses. The system includes a robust FastAPI backend and a high-performance Next.js 16 frontend for both users and administrators.
